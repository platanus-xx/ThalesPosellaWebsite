Klimbim | Don Ross (Cover by Thales Posella)


This is a cover from a beautiful song by Don Ross.
It features a Blueridge guitar with 2 large diaphram condensers.




M.A.T. | Thales Posella


This features a custom Strat straight into a Mesa Boogie Mark IV going into a Laney 4x12-v30s cabnet.
Mic setup was one sm57 On-center, another SM57 Off-center and a MD421 On-center.




Sisters | Steve Vai (Cover by Thales Posella)


This is a cover from the amazing Steve Vai.
It features an Ibanez JEM 77FP 1989 straight into a Mesa Boogie Mark V going into a Laney 4x12-v30s cabnet.
Mic setup was 2 sm57s, one On-enter and another Off-center. With an additional large diaphram condenser as room mic.




Wicked Game | Chris Isaack (Piano version by Thales Posella)


This is a piano version I've made as a tribute to this beautiful song by Chris Isaack.
It features an original Steinwein 1929 upright piano.
Mic setup was 2 small diaphram condensers on the piano and one large diaphram condenser on vocals.




You ain't the first | Guns n' Roses (Cover by Thales Posella)


This is an acoustic arrangement for this awesome song.
It features a Blueridge guitar, one small and one large diaphram condenser on the guitar and one large diaphram condenser on vocals.




Home Is Where We Belong | Thales Posella


This is an guitar-driven-instrumental song I wrote for my parents.
It features a Gibson Les Paul straight into the computer.




The Tie and The Sign | The Mavens Telescope

This is an original music project called The Mavens Telescope.
I've composed, produced, engineered, recorded all guitars, piano, synths, virtual instruments, backing vocals and mixed the whole album.
It was mastered by Andy VanDette in NYC.
